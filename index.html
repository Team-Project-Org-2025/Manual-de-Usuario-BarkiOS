Manual de Usuario

El presente manual tiene como objetivo guiar y enseñar al personal de la tienda para que puedan aprender a moverse libremente por el sistema y entender a profundidad como funcionan los módulos administrativos del mismo.

1. Inicio

El dashboard muestra todas las características generales del sistema, incluyendo métricas de ventas, compras, procesos pendientes y gráficos de gran utilidad. También se pueden imprimir hojas con toda esta información.

2. Productos

El objetivo de este módulo es poder registrar prendas con toda su información necesaria como la foto, los códigos, nombres, categorías y sus precios. Para ello deberá rellenar un formulario para cubrir toda la información y el sistema podrá asignar sus atributos.

3. Proveedores

Para los proveedores se debe rellenar un formulario similar al de productos, pero con campos específicos como RIF, nombre del comercio y dirección.

4. Clientes

Este módulo permite llevar un registro de los clientes con su información de contacto, lo cual es útil para futuras promociones o lanzamientos de productos.

5. Empleados

Este módulo permite llevar un registro del personal, incluyendo su información de contacto, tiempo en la empresa y responsabilidades.

6. Usuarios

En este módulo podrás registrar a todas aquellas personas que necesiten acceso al sistema y asignarles los permisos correspondientes según sus responsabilidades.

7. Cuentas por Cobrar

Este módulo muestra una lista de clientes con pagos pendientes, permitiendo hacer seguimiento a los créditos otorgados y su estado de pago.

8. Cuentas por Pagar

Este módulo permite llevar el control de las deudas con proveedores y otros acreedores, facilitando la gestión de pagos pendientes.

9. Ventas

Este módulo permite registrar las ventas realizadas, incluyendo información sobre los productos vendidos, montos y métodos de pago.

10. Compras

Este módulo permite registrar las compras realizadas a proveedores, incluyendo información sobre los productos adquiridos, precios y condiciones de pago.
